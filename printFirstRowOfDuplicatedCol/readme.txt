This program is intended to provide unique value for columns contain duplicated values. Users can specify which column for the duplicated columns. 
and the program only works for csv files. 

For example, the follow table: 

11-1, Guang, abc.com 
10-1, Nick, bbc.com 
11-3, Guang, ddd.com 

by running this problem, with the file as input, and pass 2 for the 2nd colum, the following will be printed out: 
11-1, Guang, abc.com 
10-1, Nick, bbc.com
